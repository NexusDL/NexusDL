# NexusDL

Interface web moderna para download de vídeos e extração de áudio de múltiplas plataformas (YouTube, TikTok, Instagram, X/Twitter). Focado em privacidade, com design responsivo.

![NexusDL Preview](https://github.com/NexusDL/NexusDL/blob/main/nexusdl_preview.png?raw=true)

## Tecnologias

- **Frontend:** Next.js 14 (App Router), React, Tailwind CSS v4.
- **UI Components:** Shadcn/ui, Lucide React.
- **Backend:** Python (FastAPI/Flask).
- **Validação:** Zod.

## Funcionalidades

- Suporte a múltiplas qualidades (360p até 4K) para YouTube.
- Remoção de marca d'água (TikTok).
- Player de vídeo integrado para pré-visualização.
- Design System consistente (Modo Claro/Escuro).
- Download direto via Proxy.
