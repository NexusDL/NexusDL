# NexusDL

A simple web interface to download videos and grab audio from platforms like YouTube, TikTok, Instagram, and X/Twitter. Built with privacy in mind, and it works fine on mobile too.

![NexusDL Preview](https://github.com/NexusDL/NexusDL/blob/main/nexusdl_preview.png?raw=true)

## Tech

* Frontend: Next.js 14 (App Router), React, Tailwind CSS v4
* UI: Shadcn/ui, Lucide React
* Backend: Python (FastAPI/Flask)
* Validation: Zod

## Features

You can pick different qualities for YouTube downloads (from 360p up to 4K).
TikTok downloads come without watermark.
There’s a built-in player if you want to preview before downloading.
Light and dark mode are both there.
Downloads go through a proxy, so it’s more direct.
